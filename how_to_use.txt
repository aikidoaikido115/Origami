All command
-help
ใช้คำสั่งนี้เพื่อดูคำสั่งทั้งหมดของ Bot
-write พิมพ์ข้อความ
ใช้คำสั่งนี้เพื่อให้ Bot พิมพ์ตามคนใช้คำสั่ง
-join
ใช้คำสั่งนี้เพื่อให้ Bot เข้ามาในห้อง
-play link เพลง
ใช้คำสั่งนี้เพื่อให้ Bot เข้าห้องและเปิดเพลง (ไม่มีระบบคิว)
-p link เพลง หรือ -p ชื่อเพลง
ใช้คำสั่งนี้เพื่อให้ Bot เข้าห้องและเปิดเพลง (ระบบคิว)
-qlist
ใช้คำสั่งนี้เพื่อดูคิวเพลง
-skip
ใช้คำสั่งนี้เพื่อสั่งให้ Bot ข้ามเพลง
-stop
ใช้คำสั่งนี้เพื่อให้ Bot หยุดเล่นเพลง (เพลงจะจบเลย)
-pause
ใช้คำสั่งนี้เพื่อให้ Bot หยุดเล่นเพลง (หยุดเล่นชั่วคราวสามารถ resume ได้)
-resume
ใช้คำสั่งนี้เพื่อให้ Bot เล่นเพลงต่อ
-discon
ใช้คำสั่งนี้เพื่อให้ Bot ออกจากห้อง
-btc
ใช้คำสั่งนี้เพื่อให้ Bot แสดงราคา Bitcoin (ดึงข้อมูลมาจาก bitkub)
-google keyword 1 + keyword 2 + keyword 3 +...+ kryword n
ใช้คำสั่งนี้เพื่อให้ Bot Search Google โดยตอนเว้นวรรคให้พิมพ์เครื่องหมาย + แทนการเว้นวรรค
